# Get the app running:

## Clone the repo using ``` git clone https://github.com/Penguin5681/DigiWell/ ```
### Then ``` cd DigiWell ```
### Run ``` yarn install ```
### ``` npx react-native run-android ``` or ``` npx react-native run-ios ```
